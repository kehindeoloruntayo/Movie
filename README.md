## Prerequisites
* _npm_
* _Installation of dotenv is required for your api key_

## Setup/Installation Requirements
* _Get a free API Key at the official [The Movie Database](https://www.themoviedb.org/settings/api)._
* _Get a free API Key at the official [The Open Movie Database](https://www.omdbapi.com/apikey.aspx)._
* _Clone the repository to your desktop._
* _Install NPM packages._
* _Enter your API in .env_
* _Navigate to the top level of the directory._
* _Open the index.html file in your browser._

 You can also read the API Documentation on the [The Movie Database](https://developer.themoviedb.org/reference/getting-started).
 You can also read the API Documentation on the [The Open Movie Database](https://www.omdbapi.com/).
